Feel

Feel is an Angular component library.

Style

is the base required for all projects, it provides all necessary css variables and automatic dark mode.

Notification

is a simple toast like notification system.

Form

provides form components, e.g. a Button, a Textfield and a Checkbox

Layout

contains utilities for placing groups of element. Have a look at this grid while changing the window width.

Usage

  1. Add@feel:registry=https://git.offsetting.dev/api/packages/offsetting/npm/ to your .npmrc.
  2. Install styling base using pnpm add @feel/style.
  3. Add node_modules/@feel/style/main.scss as an additional stylesheet in your angular.json
  4. Add node_modules/@feel/style/global to the global style imports.
  5. Include@import "theme"; @include theme();into your base stylesheet.
